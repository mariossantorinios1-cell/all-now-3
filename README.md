# AllNow - Full Final (Supabase) - Delivery

This ZIP contains the FULL AllNow scaffold prepared with Next.js (app router), TailwindCSS and Supabase placeholders.

## Important - environment variables (set in Vercel or .env.local)
- NEXT_PUBLIC_SUPABASE_URL
- NEXT_PUBLIC_SUPABASE_ANON_KEY
- NEXT_PUBLIC_GOOGLE_ANALYTICS_ID (optional)
- NEXT_PUBLIC_ADSENSE_CLIENT (optional)
- NEXT_PUBLIC_STRIPE_PUBLISHABLE_KEY (optional)
- STRIPE_SECRET_KEY (set securely in Vercel only)

## Supabase SQL to create minimal tables (run in Supabase SQL editor)
```sql
create table if not exists public.messages (
  id bigint generated by default as identity primary key,
  user_id uuid,
  username text,
  text text,
  created_at timestamptz default now()
);

create table if not exists public.views (
  id bigint generated by default as identity primary key,
  content_id text,
  user_id uuid,
  created_at timestamptz default now()
);

create table if not exists public.clicks (
  id bigint generated by default as identity primary key,
  target text,
  user_id uuid,
  created_at timestamptz default now()
);
```

## How to run locally
1. Unzip, `npm install`, create `.env.local` with the env vars, then `npm run dev`
2. For production deploy, upload to GitHub or Zip import to Vercel and set env vars, build command `npm run build`
